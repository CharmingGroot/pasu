    {{ .Values.cgroupPath }}

Make sure your agent pods actually run under that cgroup (e.g. via a
RuntimeClass or a pod cgroup_parent). Anything egressing from there that is not
allow-listed is dropped by the kernel.

{{- else }}

Tip: set `adminSocket` to enable live inspection + allowlist edits from the UI.
{{- end }}

⚠  This is a security tool running in the kernel — verify cgroupPath is a
